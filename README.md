portal-admin-tools
=========
portal-admin-tools es un módulo de Python que permite apoyar tareas administrativas en el portal (ArcGIS Online/ArcGIS Enterprise).  Estas herramientas hacen uso del [ArcGIS API for Python](https://developers.arcgis.com/python/), y el uso de [API REST](https://developers.arcgis.com/rest/users-groups-and-items/search.htm).

## Features
* Generación de Reporte (archivo csv) de contenido del portal por usuario.
* Generación de Reporte (archivo csv) de los servicios usando Carpeta de Configuración de Site.


## Requerimientos
* Python 3.x
* Trabaja con ArcGIS Online y ArcGIS Enterprise
